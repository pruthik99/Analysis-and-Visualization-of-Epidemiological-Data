# Descriptive analysis and Visualization of Epidemiological Data

## Overview

This Python project encompasses a wide array of epidemiological information for Germany. It covers essential aspects like contact settings, school data, mobility trends, and more. The data provided holds the potential to offer profound insights into disease transmission patterns and the dynamics of public health.

## Installation

1)Download the Project Folder: Begin by downloading the project folder.

2)Check Python Installation: Make sure that you have Python already installed on your system. You can check this by opening your terminal or command prompt and typing:

```
python --version

```

If Python is installed, you'll see the version number.

3)Open Terminal or Command Prompt: Launch your system's terminal (macOS and Linux) or command prompt (Windows).

4)Navigate to Project Folder: Use the "cd" command to navigate to the directory where you downloaded the project folder. For example, if you saved the folder on your desktop, you'd type:

```
cd ~/Dowanloads/project-folder

```

5)Install Required Libraries: Install the necessary libraries by executing the following command. This will automatically install the required libraries listed in the "requirements.txt" file:

```
pip install -r requirements.txt

```

This step ensures that all the dependencies are properly set up for the project.

6)Run the Project: Once the libraries are installed, start the project by entering the following command:

```
python main.py

```

This command runs the "main.py" script, which initializes the project.

7)Access the Project: After running the command, you'll see a localhost link displayed in the terminal. Click on this link to access the project's homepage in your default web browser. This is where you can interact with the project.



## Usage

This project introduces a comprehensive dashboard designed to present visualizations and statistical insights extracted from epidemiological data. Users gain access to an extensive range of information, including trends in contact settings (such as households, workplaces, and educational institutions), health statistics encompassing all-cause deaths, morbidities, and COVID-19 metrics. The dashboard offers meticulously documented data sources, definitions of variables, and insights into the data generation process. With the capacity to both evaluate and visualize this data, users are empowered to make informed decisions and remain current with the prevailing epidemiological landscape.
